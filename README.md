# MP3-Mate

YouTube to MP3/MP4 Converter

## Setup

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm run dev
```

Requires [yt-dlp](https://github.com/yt-dlp/yt-dlp):
```bash
pip install yt-dlp
```

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Express + yt-dlp
